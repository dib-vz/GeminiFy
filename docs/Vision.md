# Visión de GeminiFy

## Misión

GeminiFy es una plataforma diseñada para gestionar de forma inteligente una colección musical basada en reglas de negocio.

Su objetivo es sustituir un proceso manual basado en Microsoft Excel por un sistema robusto, auditable y escalable.

## Objetivos

- Consolidar automáticamente nuevas listas.
- Aplicar el 100 % de las reglas de negocio.
- Mantener el histórico completo.
- Auditar todas las operaciones.
- Exportar una nueva Lista Maestra compatible con Excel.
- Garantizar resultados deterministas.

## Principios

1. Nunca modificar datos sin dejar trazabilidad.
2. Todas las reglas deben estar implementadas como código.
3. Cada consolidación debe producir siempre el mismo resultado con los mismos datos de entrada.
4. La auditoría es obligatoria.
5. La Lista Maestra es la única fuente oficial de datos.
