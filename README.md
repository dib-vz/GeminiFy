# GeminiFy

## Descripción

GeminiFy es una plataforma para la gestión inteligente de listas musicales.

Su objetivo es sustituir el proceso manual basado en Excel por un sistema capaz de:

- Consolidar automáticamente nuevas listas.
- Aplicar todas las reglas de negocio.
- Auditar cada consolidación.
- Mantener el histórico completo.
- Generar una nueva Lista Maestra.
- Garantizar la trazabilidad de todos los cambios.
